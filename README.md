# LondonPost Frontend Demo

A modern news / magazine frontend built with:
- Vite
- React
- TypeScript
- Tailwind CSS
- React Router

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Included pages
- Home
- About landing
- Our Team
- Our Researchers
- What We Do
- Contact Us
- Regions landing
- Asia / Europe / Americas pages
- Dynamic country/category region view
- More section dynamic pages
- Membership
- Live Podcast
- Mission & Vision

## Notes
- All content currently uses dummy data from `src/data/siteData.ts`.
- The homepage hero uses a demo video placeholder.
- Search input is UI-only for now.
- Same-page reading sections are implemented on home, region and section pages.
